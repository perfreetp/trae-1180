## 1. 产品概述
集市管理桌面客户端，面向农贸市场办公室工作人员，用于管理固定摊位的日常经营秩序。系统涵盖摊位台账、合同费用、商户档案、进货登记、投诉处理、巡查整改、看板报表七大业务模块，实现从摊位分配、合同签约、费用收缴、日常巡查到数据报表的全流程数字化管理。

- 目标用户：农贸市场办公室管理员
- 核心价值：替代纸质台账和手工报表，提升摊位管理效率，降低欠费风险，规范经营秩序

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 市场管理员 | 系统内置账号 | 全部功能：摊位管理、合同费用、商户档案、进货登记、投诉处理、巡查整改、看板报表 |

### 2.2 功能模块
1. **摊位台账**：摊位基础信息管理、面积录入、空置摊位查询、摊位证打印
2. **合同费用**：租赁合同管理、押金登记、收费单生成、缴费状态跟踪、收费欠款统计
3. **商户档案**：商户联系人维护、营业证照上传、商户信息查询
4. **进货登记**：每日进货来源登记、冷藏商品标记、进货记录查询
5. **投诉处理**：顾客投诉受理、处理人分派、投诉状态跟踪
6. **巡查整改**：巡查问题记录、整改期限下发、复查结果跟踪
7. **看板报表**：空置摊位查询、收费欠款统计、月度经营表导出、数据看板

## 3. 核心流程

### 3.1 摊位租赁全流程
管理员在摊位台账中查看空置摊位 → 在商户档案中创建/查找商户 → 在合同费用中新建租赁合同 → 生成收费单 → 跟踪缴费状态 → 打印摊位证

### 3.2 日常经营秩序管理
商户每日登记进货来源 → 市场巡查发现并记录问题 → 下发整改通知 → 商户整改 → 复查确认 → 顾客投诉受理 → 分派处理人 → 处理完成

### 3.3 数据报表
系统自动汇总各模块数据 → 在看板报表中展示关键指标 → 管理员查询空置摊位/欠款信息 → 导出月度经营表

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深墨绿 (#1B4332) + 翡翠绿 (#2D6A4F)，象征农贸市场的生机与秩序
- 辅助色：暖琥珀 (#D4A373) 用于警示和强调，象牙白 (#FAF3E0) 作为背景
- 按钮风格：圆角矩形，主操作使用实心翡翠绿，次要操作使用描边样式
- 字体：思源宋体/Noto Serif SC 用于标题（展现传统市场韵味），Noto Sans SC 用于正文
- 布局风格：左侧固定导航栏 + 右侧内容区，表格为主的数据展示，卡片式统计指标
- 图标风格：线性图标，统一2px描边，来自Lucide图标库

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 全局布局 | 侧边导航 | 深墨绿背景，7个导航项带图标，当前项高亮 |
| 全局布局 | 顶部栏 | 显示当前模块名称、搜索框、用户头像 |
| 摊位台账 | 摊位列表 | 数据表格 + 筛选栏 + 新增按钮，卡片式空置摊位展示 |
| 合同费用 | 合同管理 | 左侧合同列表 + 右侧详情面板，收费单弹窗编辑 |
| 商户档案 | 商户详情 | 头部信息卡 + 标签页切换（基本信息/证照/关联摊位） |
| 进货登记 | 进货记录 | 日期选择器 + 数据表格，冷藏标记用蓝色标签区分 |
| 投诉处理 | 投诉列表 | 看板式布局（待处理/处理中/已完成三列） |
| 巡查整改 | 巡查记录 | 时间线式展示 + 表格视图切换 |

### 4.3 响应式设计
- 桌面优先设计，最小支持1280px宽度
- 侧边导航可折叠为图标模式
- 表格支持横向滚动
- 弹窗居中显示，最大宽度640px
